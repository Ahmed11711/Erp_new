# فلو تأكيد الطلب - order_flow

## إعداد القالب في WhatsApp Manager (Meta)

عند إنشاء القالب `order_flow` في Meta Business Manager، يُفضّل تعيين **معرّفات الأزرار** (Button IDs) كما يلي (أو الاعتماد على **نص الزر** المعروض — الكود يطابق النص العربي أيضاً):

| نص الزر (كما يظهر للعميل) | Button ID (مقترَح) | الإجراء في النظام |
|---------------------------|---------------------|---------------------|
| إعادة الشحن               | `confirm_order` أو `reship` | تأكيد / متابعة الشحن |
| تعديل الطلب               | `postpone_order` أو `modify_order` | فلو التأجيل |
| إلغاء الطلب               | `cancel_order`      | فلو الإلغاء |

**ملاحظة:** نوع كل زر يجب أن يكون **رد سريع (Quick Reply)**. إذا غيّرت معرّفات ميتا، تأكد أن **نص الزر** يطابق أحد الأعمدة أعلاه أو يحتوي «إعادة الشحن» / «تعديل…طلب» / «إلغاء…طلب» لأن النظام يطابق النص احتياطياً.

### دعم القوالب الموجودة

إذا كان القالب يحتوي على أزرار بأسماء مختلفة، يدعم النظام الربط التلقائي:

| نص الزر        | الإجراء |
|----------------|---------|
| تأكيد الطلب / إعادة الشحن | تأكيد الطلب |
| تأجيل الطلب / تعديل الطلب | تأجيل الطلب |
| إلغاء الطلب    | إلغاء الطلب |

## الفلو (شات فقط)

**لا يُعدّل النظام حالة الطلب أو بياناته** عند الضغط على الأزرار — الردود واتساب إعلامية فقط (`OrderConfirmationFlowService`). النصوص أدناه هي المرجع في الكود.

### 1) تأكيد الطلب (`confirm_order`)

**Message 2**

```
شكرًا لك 🙌
تم تأكيد طلبك وسيتم التواصل معك قبل الشحن.

📦 موعد الشحن المتوقع:
{{shipping_time}}   ← من delivery_date أو نص افتراضي

فريق Magalis في خدمتك دائمًا.
```

### 2) تأجيل الطلب (`postpone_order` → جلسة)

**Message 1**

```
تمام 👍
هل تود تأجيل الطلب؟

اختر الوقت المناسب لك:

1️⃣ بعد يومين
2️⃣ بعد أسبوع
3️⃣ تحديد موعد آخر
```

**Message 2** (بعد اختيار 1/2/3)

```
تم تأجيل طلبك بنجاح ✔️
وسنقوم بالتواصل معك في الموعد المحدد لتأكيد الشحن.

رقم الطلب: #{{order}}
```

### 3) رفض الاستلام (`delivery_rejection_start` → جلسة)

**Message 1**

```
لاحظنا أنه تم رفض استلام الطلب #{{order}} عند التسليم.

نحب نتأكد فقط من السبب حتى نقدر نساعدك:

1️⃣ أريد إعادة الشحن
2️⃣ أريد تعديل الطلب
3️⃣ أريد إلغاء الطلب
```

**إذا اختار إعادة الشحن** (`reject_reship`)

```
لا مشكلة 👍
سنقوم بترتيب إعادة شحن الطلب مرة أخرى.

سيتم التواصل معك لتأكيد العنوان وموعد التسليم.
```

(الخياران 2 و 3: يبقى نص التواصل مع خدمة العملاء كما في الكود.)

### 4) إلغاء الطلب (`cancel_order` → جلسة)

**Message 1**

```
هل أنت متأكد من إلغاء الطلب #{{order}}؟

1️⃣ نعم إلغاء الطلب
2️⃣ لا أريد تأكيد الطلب
```

**إذا أكد الإلغاء** (`cancel_yes_confirm`)

```
تم إلغاء طلبك بنجاح.

يسعدنا خدمتك مرة أخرى في أي وقت 🤍
فريق Magalis
```

إن وُجدت جلسات قديمة من فلو تفاعلي سابق، تُغلق دون تعديل الطلب.

## عرض القوالب حسب الرقم

في واجهة إرسال الرسائل، يمكن اختيار رقم واتساب لعرض القوالب التابعة له فقط (عند تعيين `phone_number_id` في `config/whatsapp_meta_templates.php`).
