# فلو تأكيد الطلب - order_confirmation_flow

## إعداد القالب في WhatsApp Manager (Meta)

عند إنشاء القالب `order_confirmation_flow` في Meta Business Manager، يجب تعيين **معرّفات الأزرار** (Button IDs) بالضبط كما يلي:

| الزر           | Button ID       |
|----------------|-----------------|
| تأكيد الطلب    | `confirm_order`  |
| تأجيل الطلب    | `postpone_order` |
| إلغاء الطلب    | `cancel_order`   |

**ملاحظة:** نوع كل زر يجب أن يكون **مخصص (Quick Reply)**.

### دعم القوالب الموجودة

إذا كان القالب يحتوي على أزرار بأسماء مختلفة، يدعم النظام الربط التلقائي:

| نص الزر        | الإجراء |
|----------------|---------|
| تأكيد الطلب / إعادة الشحن | تأكيد الطلب |
| تأجيل الطلب / تعديل الطلب | تأجيل الطلب |
| إلغاء الطلب    | إلغاء الطلب |

## الفلو

1. **الرسالة الأولى (قالب):** تُرسل عند وصول الطلب وتجهيزه.
2. **تأكيد الطلب:** تحديث حالة الطلب إلى "طلب مؤكد" وإرسال رسالة تأكيد الشحن.
3. **تأجيل الطلب:** عرض خيارات (بعد يومين، بعد أسبوع، تحديد موعد آخر) ثم تأجيل الطلب.
4. **إلغاء الطلب:** طلب تأكيد الإلغاء ثم إلغاء الطلب أو التراجع.

## عرض القوالب حسب الرقم

في واجهة إرسال الرسائل، يمكن اختيار رقم واتساب لعرض القوالب التابعة له فقط (عند تعيين `phone_number_id` في `config/whatsapp_meta_templates.php`).
